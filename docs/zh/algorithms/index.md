# 压缩算法

fqc 使用多种压缩算法来处理不同类型的 FASTQ 数据。

## 算法概览

### ABC 算法（短读长）
- 适用于 < 300bp 的短读长序列
- 使用一致性序列 + 增量编码
- 高质量分数压缩

### Zstd（中长读长）
- 适用于中等和长读长序列
- 通用压缩算法
- 快速解压速度

### SCM 质量分数压缩
- 统计上下文模型
- 算术编码
- 有损和无损模式

## 实现细节

算法实现在 `src/algo/` 目录中：

- `block_compressor.rs` - 主要压缩器
- `dna.rs` - DNA 编码表
- `quality_compressor.rs` - 质量分数压缩
- `id_compressor.rs` - ID 压缩
- `pe_optimizer.rs` - 双端优化

## 性能比较

| 算法 | 压缩率 | 速度 | 适用场景 |
|------|--------|------|----------|
| ABC | 最高 | 中等 | 短读长 (< 300bp) |
| Zstd | 高 | 快速 | 中长读长 |
| SCM | 最高 | 中等 | 质量分数 |
