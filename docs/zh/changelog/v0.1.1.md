# 版本 v0.1.1

**发布日期**: 2026-04-16

## 发布说明

这是一个维护性发布，专注于文档改进、工作流增强和安全加固。

---

## 新增内容

### 📚 文档

#### GitBook 增强
- **SECURITY.md** — 新增安全策略文档，包含漏洞报告指南
- **GitBook 词汇表** — 新增 `GLOSSARY.md`，解释 ABC、SCM、PE 等领域术语
- **GitBook 插件** — 新增 `anchors` 和 `search-pro` 插件，改善导航体验
- **PDF 设置** — 在 `book.json` 中配置 GitBook PDF 输出设置
- **侧边栏链接** — 在导航中添加 GitHub、Releases 和 Issues 的外部链接

#### 性能文档
- 修复性能文档中错误的默认压缩级别（从 3 更正为 6）

### 🔧 CI/CD 改进

#### 新功能
- **PR 预览** — 在 `pages.yml` 中为拉取请求提供文档预览
- **Trivy 安全扫描** — 在 `docker.yml` 中增加 Docker 镜像漏洞扫描
- **SHA512 校验和** — 发布产物同时提供 SHA256 和 SHA512 校验和
- **CI 汇总任务** — 合并 CI 状态报告，提供统一视图
- **文档检查任务** — 在质量工作流中新增 `doc-check` 任务

#### 工作流增强
- **Pages 工作流** — 修复缺失的 `configure-pages` 步骤并添加构建验证
- **Docker 工作流** — 添加 `security-events` 权限以支持 Trivy 扫描
- **CI 工作流** — 失败时收集测试日志便于调试
- **质量工作流** — 添加质量门汇总和文档检查

### 📦 依赖与构建

- **package.json** — 新增 `docs:clean` 和 `docs:check` 脚本，并指定 Node.js 引擎要求

---
