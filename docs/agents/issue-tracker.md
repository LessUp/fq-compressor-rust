# Issue tracker: GitHub

Issues for this repo live in GitHub Issues.

## Location

https://github.com/LessUp/fq-compressor-rust/issues

## CLI

Use the `gh` CLI for all issue operations:

- `gh issue create` — create a new issue
- `gh issue view <number>` — view an issue
- `gh issue list` — list issues
- `gh issue edit <number>` — edit an issue
- `gh issue close <number>` — close an issue
- `gh issue reopen <number>` — reopen an issue

## Labels

Labels are applied via `gh issue edit <number> --add-label <label>`.

See `triage-labels.md` for the canonical label vocabulary.
