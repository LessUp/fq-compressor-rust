# project-governance change

## ADDED Requirements

### Requirement: stale repository artifacts must be removed aggressively

The repository MUST prefer deletion or rewrite over preservation when process files, docs, or automation are low-value or misleading.

#### Scenario: a contributor finds obsolete repository scaffolding

- **WHEN** they confirm the file is no longer trustworthy or useful
- **THEN** they remove or replace it in the same cleanup change
