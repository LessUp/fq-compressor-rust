# Code of Conduct

`fqc` is a small Rust FASTQ compression project. Keep discussions technical, respectful, and focused on improving the code, format documentation, validation, and release quality.

Unacceptable conduct includes harassment, personal attacks, publishing private information, or intentionally derailing issue and review threads. Maintainers may edit, close, or remove contributions that undermine a professional engineering environment.

For security-sensitive reports, do not use public issues. Follow [`SECURITY.md`](SECURITY.md) and use GitHub private vulnerability reporting.
