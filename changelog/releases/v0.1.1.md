# Release v0.1.1

**Release Date**: 2026-04-16

## Release Notes

This is a maintenance release focusing on documentation improvements, workflow enhancements, and security hardening.

---

## What's New

### 📚 Documentation

#### GitBook Enhancements
- **SECURITY.md** — Added security policy with vulnerability reporting guidelines
- **GitBook Glossary** — New `GLOSSARY.md` with terminology definitions for ABC, SCM, PE, and other domain terms
- **GitBook Plugins** — Added `anchors` and `search-pro` for improved navigation experience
- **PDF Settings** — Configured GitBook PDF output settings in `book.json`
- **Sidebar Links** — Added external links to GitHub, Releases, and Issues in navigation

#### Performance Documentation
- Fixed incorrect default compression level (changed from 3 to 6) in performance docs

### 🔧 CI/CD Improvements

#### New Features
- **PR Preview** — Documentation preview for pull requests in `pages.yml`
- **Trivy Security Scanning** — Docker image vulnerability scanning in `docker.yml`
- **SHA512 Checksums** — Added to release artifacts alongside SHA256
- **CI Summary Job** — Consolidated CI status reporting with job summaries
- **Documentation Check** — New `doc-check` job in quality workflow

#### Workflow Enhancements
- **Pages Workflow** — Fixed missing `configure-pages` step and added build validation
- **Docker Workflow** — Added `security-events` permission for Trivy scanning
- **CI Workflow** — Added test log artifacts on failure for debugging
- **Quality Workflow** — Added quality gate summarization

### 📦 Dependency & Build

- **package.json** — Added `docs:clean` and `docs:check` scripts with Node.js engine requirement

---

## Migration Guide

No breaking changes. Users can upgrade seamlessly from v0.1.0.

### For Users

```bash
# Pull latest Docker image
docker pull ghcr.io/lessup/fq-compressor-rust:v0.1.1

# Or download binary from GitHub Releases
# https://github.com/LessUp/fq-compressor-rust/releases/tag/v0.1.1
```

### For Contributors

No changes required to development workflow. All 131 tests continue to pass.
